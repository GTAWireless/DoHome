## 2. 产品说明
本款产品内置多个传感器，包括人体红外，光照度，和温湿度，可以对当前室内环境中的光照强度，温度以及湿度，以及是否有人员走动进行实时动态的检测，同时内置继电器模组，可外接对其它优点设备进行有效控制，其中外壳往逆时针旋转到对应刻度线即可拆下。
