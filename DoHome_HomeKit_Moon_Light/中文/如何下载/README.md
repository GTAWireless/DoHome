## 1. 如何烧写固件（windows）
1. 安装[串口驱动](http://www.usb-drivers.org/ft232r-usb-uart-driver.html)
2. 安装[烧写工具](http://espressif.com/en/support/download/other-tools)
3. 烧录相关 bin 文件
选择合适的串口后，在按照如下图所示，配置串口号、串口波特率等，按 START 即可开始下载程序。
