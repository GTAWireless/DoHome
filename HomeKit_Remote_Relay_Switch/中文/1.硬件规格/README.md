## 2.远程继电器开关接口说明
### 1.接口说明

继电器上共有五个可外接进行控制的接口，从左到右依次为1.5V电源口，2.GND口，3.常闭NC口，4.共用CON口，5.常开NO口,如下图所示。

  <img src="../README_IMAGE/2.png" width="400" />
  
### 2.使用介绍

当外接常开NO口和公共CON口时，手机端打开开关，继电器通电闭合，NO与COM导通，手机端关闭，继电器断开，NO与CON断开;外接常闭口NC和公共口CON时则与此相反。
